运行环境

  1. Python 3.x.x
  2. 需要使用的第三方库在requirements.txt，如下：
  • numpy
  • imutils
  • sklearn
  • tensorflow
  • keras
  • opencv2-python
pip3 install -r requirements.txt

解压

  • 将gen_captcha.7z解压

提取

  • 从验证码图像中提取单个字符
python extract_single_letters_from_captchas.py

提取出来的单个字符集会包含在ext_letter这个文件夹中
